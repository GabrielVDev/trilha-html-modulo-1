Tags Principais

Títulos e Parágrafos

Tag h1 a h6 = Essas tags são usadas para definir cabeçalhos, onde h1 é o maior e mais importante, e h6 é o menor e menos importante.

Tag P = Define um parágrafo de texto.

Formatação de Texto

Tag mark = Destaca o texto, geralmente com um fundo amarelo.

Tag samll = Define texto pequeno.

Tag i = Define texto em itálico.

Tag u = Define texto sublinhado.

Tag strong = Define texto em negrito e com importância.

Listas

Tag ol = Define uma lista ordenada

  1. Primeiro item
  2. Segundo item
  3. Terceiro item

Tag ul = Define uma lista não ordenada (com marcadores).

  • Primeiro item
  • Segundo item
  • Terceiro item

Tag li = Define um item de lista.

Links e Linhas Horizontais

Tag a = Define um link. Visite o google

Tag hr = Insere uma linha horizontal.

Subscrito e Sobrescrito

Tag sub = Define texto subscrito (abaixo da linha base).

Tag sup = Define texto sobrescrito (acima da linha base).

Citações

Tag blockquote =

Define uma seção de citação em bloco.

Tags Complementares

Tag font = Esta tag era usada para definir a cor, fonte e tamanho do texto. No entanto, ela está obsoleta no HTML5. Em vez disso, recomenda-se usar CSS para estilizar o texto.

Tag del = Define texto que foi removido ou deletado de um documento.O texto aparece riscado.

Tag abbr = Define uma abreviação ou acrônimo.
